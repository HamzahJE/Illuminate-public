# Illuminate
Wearable AI headgear providing real-time feedback for visually impaired users via audio/visual outputs and voice/button controls.


1. pip install dotenv
2. pip install openai


Please create a .env in the same directory as your script with the following:

MODEL=gpt-4o \
OPENAI_API_BASE=YOUR_BASE_URL \
OPENAI_API_KEY=X \
OPENAI_ORGANIZATION=X \
API_VERSION=YOUR_API_VERSION 
IMAGE_PATH=path_to_images_folder
